# 猎聘岗位样例

岗位：高级解决方案架构师 - AI Agent 方向

薪资：60-90 万/年

地点：上海 / 可接受短期出差

公司：某头部企业服务 SaaS 厂商

职责：
- 面向大型客户，负责 AI Agent、知识库、流程自动化相关解决方案设计；
- 与销售、产品、研发、交付团队协作，完成售前交流、方案设计、POC 推进和项目复盘；
- 基于行业客户需求，沉淀可复制的行业解决方案和标杆案例；
- 对客户高层进行产品价值讲解，识别关键业务痛点。

要求：
- 5 年以上 ToB / SaaS / 解决方案 / 售前经验；
- 熟悉大模型、RAG、Agent、低代码、企业系统集成等方向；
- 有制造、零售、金融、互联网等行业客户经验优先；
- 能适应一定频率客户现场沟通。
