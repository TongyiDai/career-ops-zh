# 模式：中文简历定制 resume

## 目标

基于 `cv.md` 和目标 JD，生成一份更适合中文市场的岗位定制简历。不要伪造经历，只能重排、改写、突出已有事实。

## 输出结构

```markdown
# {姓名} - {目标岗位} 定制简历

## 个人摘要
3-5 行，紧贴 JD，不要空泛。

## 核心优势
- ...

## 工作经历
### 公司｜岗位｜时间
- 动作 + 复杂度 + 方法 + 结果。

## 代表项目
### 项目名称
- 背景：
- 角色：
- 做法：
- 结果：
- 与目标岗位的关联：

## 技能关键词
- ...
```

## 中文简历原则

- 讲清职责边界；
- 讲清业务规模；
- 讲清协作对象；
- 讲清可量化结果；
- 讲清方法论沉淀；
- 不堆砌形容词；
- 不编造指标。
